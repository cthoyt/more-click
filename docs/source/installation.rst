Installation
============

The most recent release can be installed from `PyPI
<https://pypi.org/project/more_click>`_ with uv:

.. code-block:: console

    $ uv pip install more_click

or with pip:

.. code-block:: console

    $ python3 -m pip install more_click

Installing from git
-------------------

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/cthoyt/more-click.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/cthoyt/more-click.git

Installing for development
--------------------------

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/cthoyt/more-click.git
    $ cd more-click
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
